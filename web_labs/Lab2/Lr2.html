Вызов функций происходит как через консоль, так и через форму

Есть 3 вызова функций через консоль:

  • RetArg(arg) - возвращает переданный аргумент
  • Summ(arg1, arg2) - возвращает сумму переданных аргументов
  • Sqr() - возвращает квадраты значений массива и начальный массив

    • Функция возврата квадрата значений массива и начального массива вызывается только из консоли
